# MCG Windows & Doors — Frequently Used WERS Codes (U-value/SHGC Chart)

## Summary

A one-page reference chart from M.C.G Windows & Doors (mcgwindows.com.au, 1300 766 349) listing frequently used WERS IDs with U-values and SHGCs for their aluminium window and door ranges, all in double glazed clear or double glazed Low-E. Useful for entering MCG window performance values into NatHERS software; U-values span 2.2–4.4 W/m²K and SHGC 0.45–0.70 across eight product types.

## Key claims

- Signature Sliding Windows: WERS BRD-125-88 double glazed Low-E U2.2 / SHGC 0.54; BRD-125-46 double glazed clear U3.2 / SHGC 0.67.
- Essential Fixed Windows: BRD-086-85 Low-E U2.7 / 0.53; BRD-086-09 clear U3.6 / 0.70.
- Signature Fixed Windows: BRD-189-011 Low-E U3.1 / 0.46; BRD-189-007 clear U3.8 / 0.58.
- Essential Awnings: BRD-082-83 Low-E U3.3 / 0.45; BRD-082-09 clear U4.4 / 0.58.
- Signature Awnings: BRD-035-55 Low-E U2.7 / 0.53; BRD-035-15 clear U3.6 / 0.66.
- Signature Sliding Doors: BRD-052-83 Low-E U3.2 / 0.48; BRD-052-11 clear U4.0 / 0.59.
- Essential Sliding Windows: BRD-154-55 Low-E U3.0 / 0.50; BRD-154-36 clear U3.8 / 0.61.
- Commercial Sliding Doors: BRD-045-46 Low-E U3.2 / 0.50; BRD-045-14 clear U4.1 / 0.62.
- Across the chart, Low-E glazing improves U-value by roughly 0.6–1.1 W/m²K and lowers SHGC by about 0.10–0.13 versus clear double glazing in the same frame.
- Units are not stated on the chart; WERS U-values are conventionally whole-window W/m²K and SHGC is dimensionless.
- This chart covers the standard (non-thermally-broken) external range; a separate companion document covers MCG's thermally broken range.

## Caveats

- Undated, marketing-format chart with no test conditions or WERS certification dates shown; confirm current WERS listings (wers.net) before locking values into an assessment.
- Window/door type-to-code pairings were reconstructed from a degraded table layout in extraction; verify pairings against the source PDF before quoting a specific product.
