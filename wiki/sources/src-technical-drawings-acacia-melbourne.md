# Acacia House Melbourne — Technical Drawings

## Summary

Architectural drawing set (single 10-sheet package) for the Acacia House single-storey reference design configured for Melbourne, covering site plan, floor plan, roof plan, four elevations, section, wall setout plan and a window/door schedule. Issued March 2020 at "Preliminary" project stage by the Commonwealth as a guide-only exemplar design; intended to be read alongside the companion Acacia House Melbourne "Design Options" document, which holds the building specifications and NatHERS rating information.

## Key claims

- Drawing set comprises 10 sheets: Site Plan (1:200), Floor Plan (1:100), Roof Plan (1:100), four Elevations (1:100), Section (1:100), Wall Setout Plan (1:100), and Window and Door Schedule — all A3 paper size, issued March 2020, project stage "Preliminary".
- Unlike the Brisbane and Hobart Acacia drawing sets (which each contain Option 1 and Option 2 packages), the Melbourne set contains a single design version only.
- The drawings explicitly state they "should be read with the Acacia House Melbourne 'Design Options' document which includes building specifications and rating information" — thermal specs and star ratings are NOT in this drawing set.
- Window/door schedule lists 17 openings: main door D1 2400mm x 1100mm solid door; glazed sliding doors SD1–SD7 all 2400mm high (widths 2000–2300mm); fixed/awning windows W2/W3/W5 2400mm x 900mm; awning window W4 1440mm x 600mm; fixed window W6 800mm x 2300mm; sliding window W7 600mm x 600mm; glazed hinged door D2 2250mm x 820mm.
- Most openings are shaded by a 1000mm eave at 2700mm above Finished Floor Level — identical schedule to the Brisbane Option 1 configuration.
- Carries a Commonwealth disclaimer: plan is a guide only, not professional advice; no responsibility accepted for accuracy, fitness for purpose or regulatory compliance.

## Caveats

- These are architectural drawings — text extraction captures only title-block metadata and the window/door schedule. The actual plans, elevations, dimensions and annotations are graphical and must be viewed in the source PDF.
- Issued March 2020 at "Preliminary" stage, predating NCC 2022 7-star requirements; rating/specification claims live in the separate "Design Options — Acacia Melbourne" document.
- Guide-only Commonwealth exemplar; not site-specific or certified for construction.
