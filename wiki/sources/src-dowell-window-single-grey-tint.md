# Dowell Windows — Single Glazed Grey Tint UW/SHGC Values by Window Type

## Summary

An email from Andrew Greenstreet (Account Manager Sunshine Coast, Dowell Windows Pty Ltd, Geebung QLD) giving project-specific UW and SHGC values for Dowell single-glazed grey tinted glass across five window/door types — all at UW 6.1, with SHGC ranging 0.44 (stand-alone louvres) to 0.55 (fixed). It also documents Dowell's workaround for coupled louvre-and-fixed windows, which have no tested rating.

## Key claims

- Context line: "Grey tinted glass has been used for this project …." — values are for single glazed grey tint.
- Louvres (stand alone): UW 6.1, SHGC 0.44.
- Fixed (stand alone): UW 6.1, SHGC 0.55.
- Louvre fixed * (coupled): UW 6.1, SHGC 0.55.
- Hinged door: UW 6.1, SHGC 0.46.
- Sliding doors: UW 6.1, SHGC 0.53.
- Footnote on the coupled unit: "* don't actually have a rating for coupled windows (lovure & fixed) - have used the poorer performing window (fixed light) at SHGC 0.55. The reality is the SHGC, if tested maybe slightly lower as part of the opening is the better performing Louvre at 0.44 SHGC" (sic).

- Source is a screenshot of a supplier email; no date is visible in the image and the project it relates to is not named ("this project") — values are project-quoted, not a published WERS datasheet.
- The coupled louvre+fixed SHGC of 0.55 is the supplier's conservative substitution (worst-performing component), not a tested rating — the email itself flags the true value is likely slightly lower.
- UW 6.1 across all types indicates standard aluminium single glazing; confirm against current Dowell/WERS data before reuse on other projects.
