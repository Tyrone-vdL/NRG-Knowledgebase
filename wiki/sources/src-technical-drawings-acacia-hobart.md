# Acacia House Hobart — Technical Drawings (Options 1 & 2)

## Summary

Architectural drawing set (two 10-sheet packages) for the Acacia House single-storey reference design configured for Hobart, covering site plan, floor plan, roof plan, four elevations, section, wall setout plan and a window/door schedule for both Option 1 and Option 2. Issued March 2020 at "Preliminary" project stage by the Commonwealth as a guide-only exemplar design; intended to be read alongside the companion Acacia House Hobart "Design Options" document, which holds the building specifications and NatHERS rating information.

## Key claims

- Drawing set comprises 10 sheets per option: Site Plan (1:200), Floor Plan (1:100), Roof Plan (1:100), four Elevations (1:100), Section (1:100), Wall Setout Plan (1:100), and Window and Door Schedule — all A3 paper size, issued March 2020, project stage "Preliminary".
- Two design options are provided for Hobart (Option 1 and Option 2), each with its own complete 10-sheet set and window/door schedule.
- The drawings explicitly state they "should be read with the Acacia House Hobart 'Design Options' document which includes building specifications and rating information" — thermal specs and star ratings are NOT in this drawing set.
- Window/door schedule (Option 1) lists 17 openings: main door D1 2400mm x 1100mm solid door; glazed sliding doors SD1–SD7 all 2400mm high (widths 2000–2300mm); fixed/awning windows W2/W3/W5 2400mm x 900mm; awning window W4 1440mm x 600mm; fixed window W6 800mm x 2300mm; sliding window W7 600mm x 600mm; glazed hinged door D2 2250mm x 820mm.
- Option 1 openings note a 1000mm eave at 2700mm above Finished Floor Level; the Option 2 schedule reduces this to a 600mm eave at 2700mm above FFL (and D1 narrows to 1000mm wide) — shallower eaves admit more winter sun, consistent with Hobart's cool-climate heating-dominated loads (contrast: Brisbane Acacia keeps 1000mm eaves in both options).
- Unlike Brisbane Option 2 (which swaps in louvres for ventilation), Hobart Option 2 retains fixed/awning windows throughout — ventilation area is not increased for the cool climate.
- Carries a Commonwealth disclaimer: plan is a guide only, not professional advice; no responsibility accepted for accuracy, fitness for purpose or regulatory compliance.

## Caveats

- These are architectural drawings — text extraction captures only title-block metadata and the window/door schedules. The actual plans, elevations, dimensions and annotations are graphical and must be viewed in the source PDF.
- Issued March 2020 at "Preliminary" stage, predating NCC 2022 7-star requirements; rating/specification claims live in the separate "Design Options — Acacia Hobart" document.
- Guide-only Commonwealth exemplar; not site-specific or certified for construction.
