# Telopea House Hobart — Technical Drawings (Options 1 & 2)

## Summary

Architectural drawing set (two 12-sheet packages) for the Telopea House two-storey reference design configured for Hobart, covering site plan, ground and first floor plans, roof plan, four elevations, section, ground/first floor wall setout plans and a window/door schedule for both Option 1 and Option 2. Issued March 2020 at "Preliminary" project stage by the Commonwealth as a guide-only exemplar design; intended to be read alongside the companion Telopea House Hobart "Design Options" document, which holds the building specifications and NatHERS rating information.

## Key claims

- Drawing set comprises 12 sheets per option: Site Plan (1:200), Ground Floor Plan, First Floor Plan, Roof Plan, four Elevations, Section, Ground Floor Wall Setout Plan, First Floor Wall Setout Plan (all 1:100), and Window and Door Schedule — A3 paper size, issued March 2020, project stage "Preliminary". The two-storey format (12 sheets) distinguishes Telopea from the single-storey Acacia (10 sheets).
- Two design options are provided for Hobart (Option 1 and Option 2), each with its own complete 12-sheet set and window/door schedule.
- The drawings explicitly state they "should be read with the Telopea House Hobart 'Design Options' document which includes building specifications and rating information" — thermal specs and star ratings are NOT in this drawing set.
- Option 1 schedule: solid doors D1 2250mm x 1000mm (800mm eave), D2/D3 2250mm x 820mm; glazed hinged door HD1 2250mm x 920mm (600mm eave); glazed sliding doors SD2/SD3 2250mm x 2850mm (600mm eave); upper-level fixed/casement windows W6–W11, W13, W14 at 1650mm x 1500mm with 600mm eaves; W2 1050mm x 1500mm fixed/casement with a 600mm sun hood; W4 2250mm x 880mm fixed window with an external sliding sun screen; sliding windows W12 (1650mm x 880mm) and W15 (1650mm x 1340mm) both with external sliding sun screens plus 600mm eaves; W16 900mm x 600mm casement, frosted.
- Option 2 changes vs Option 1: HD1 becomes a glazed sliding door, an additional sliding door SD4 2250mm x 2850mm is added, W2 widens to 2850mm, W3 is deleted, and the upper-level casement/sliding windows increase in height from 1650mm to 1800mm — i.e. Option 2 carries noticeably more glazed area.
- Shading devices are a mix of 600–800mm eaves, a 600mm sun hood, and external sliding sun screens on selected windows — adjustable external shading appears on both options.
- Carries a Commonwealth disclaimer: plan is a guide only, not professional advice; no responsibility accepted for accuracy, fitness for purpose or regulatory compliance.
