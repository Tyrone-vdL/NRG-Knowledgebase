# ABCB Understanding the NCC — Building Classifications (Class 1-10)

## Summary

ABCB "Understanding the NCC" guidance sheet (© 2017) explaining all NCC building classifications Class 1 through Class 10, including sub-classifications (1a/1b, 7a/7b, 9a/9b/9c, 10a/10b/10c), sole-occupancy units, mixed-use buildings and the 10% ancillary-use rule. Useful as the reference for deciding whether a dwelling is Class 1a, 1b, 2 or a Class 10 structure — the gateway question for which NCC volume and energy provisions apply.

## Key claims

- Class 2-9 buildings are mostly covered by NCC Volume One; Class 1 and 10 buildings are mostly covered by NCC Volume Two; Volume Three (Plumbing Code of Australia) refers to all classifications. Classification provisions sit in Volume One Part A3, Volume Two Part 1.3 and Volume Three Part A4.
- Class 1a: a single dwelling — a detached house, or one of a group of attached dwellings (town house, row house or the like). Attached Class 1 buildings must be separated by a wall with fire-resisting and sound insulation properties.
- Class 1 buildings cannot be located above or below any other dwelling or any other class of building, other than a private garage.
- Class 1b: a boarding house, guest house or hostel with floor area less than 300 m² and ordinarily fewer than 12 residents; or four or more single dwellings on one allotment used for short-term holiday accommodation.
- Class 2: apartment buildings — multi-unit residential where people live above and below each other; can also be single-storey attached dwellings with a common space (e.g. basement/carpark) below.
- Class 3: residential buildings other than Class 1 or 2 — boarding houses/hostels/backpackers above Class 1b limits, dormitories, workers' quarters, and residential parts of hotels, motels, schools, hospitals or jails.
- Class 4: a single dwelling within a non-residential (Class 5-9) building, e.g. a caretaker's residence; only one Class 4 part permitted per building.
- Class 5: offices (lawyers, accountants, GPs, government, architects); Class 6: shops, restaurants, cafés, service stations; Class 7a: carparks; Class 7b: warehouses/wholesale display; Class 8: factories, workshops, laboratories.
- Class 9a: health-care buildings (hospitals); Class 9b: assembly buildings (schools, universities, childcare, sporting facilities, nightclubs, public transport buildings); Class 9c: aged care buildings with 24-hour staff assistance.
- Class 10a: non-habitable buildings — sheds, carports, private garages; Class 10b: structures — fences, masts, antennas, retaining walls, swimming pools; Class 10c: private bushfire shelters associated with (but not attached to) a Class 1a building.
- A private garage is: a garage associated with a Class 1 building; or a single storey of a building containing not more than 3 vehicle spaces; or a separate single-storey garage associated with another building containing no more than 3 vehicles.
- 10% ancillary rule: a part of a building with a different purpose that is not more than 10% of the floor area of the storey it is on may be treated as ancillary to the major use rather than separately classified.
- A Sole Occupancy Unit (SOU) is a part of a building occupied by an owner, lessee or tenant to the exclusion of others — e.g. an apartment, a hotel suite, a shop in a shopping centre.
- Buildings can hold multiple classifications (e.g. Class 5/6/7) where designed to meet the minimum requirements of all of them; where classification is unclear, the approval authority has discretion to decide.

## Caveats

- © 2017 publication, so clause references reflect pre-2022 NCC structure (e.g. Volume Two Part 1.3); under NCC 2022 the classifications sit in Part A6 of the Governing Requirements, though the class definitions themselves are substantively unchanged.
- National perspective only — does not address State or Territory variations.
- Guidance material only; not a substitute for the NCC or State/Territory legislation.
