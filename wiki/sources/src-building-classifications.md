# ABCB Understanding the NCC — Building Classifications (2017)

## Summary

ABCB "Understanding the NCC" guidance sheet (© 2017) explaining the NCC building classification system Class 1 through Class 10, including sub-classifications, sole-occupancy units, mixed-use and multiple classifications, and the 10% ancillary-use rule. This is the plain-English national reference for deciding which class a building (or part) falls into, which in turn determines which NCC volume and which energy efficiency provisions apply.

## Key claims

- Classifications group buildings by function and use; Class 2–9 buildings are mostly covered by NCC Volume One, Class 1 and 10 by Volume Two; Volume Three (PCA) refers to all classes. Classification references: Vol One Part A3, Vol Two Part 1.3, Vol Three Part A4 (as at 2017).
- Class 1a: a single dwelling — detached house, or one of a group of attached dwellings (terrace/row/town house) separated by walls with fire-resisting and sound insulation properties. Class 1 buildings cannot be located above or below any other dwelling or class of building other than a private garage.
- Class 1b: a boarding house, guest house or hostel with floor area less than 300 m² and ordinarily fewer than 12 residents; or four or more single dwellings on one allotment used for short-term holiday accommodation.
- Class 2: apartment buildings — multi-unit residential where people live above/below each other; can also be single-storey attached dwellings above a common space (e.g. basement/carpark). Class 3: residential buildings other than Class 1 or 2 for long-term or transient living by unrelated persons (boarding houses above 1b limits, hostels, backpackers, residential parts of hotels/motels/schools/hospitals/jails, some care-type accommodation).
- Class 4: a dwelling within a non-residential (Class 5–9) building, e.g. a caretaker's residence; only one Class 4 part is permitted per building, and it cannot be in a Class 1, 2 or 3 building.
- Class 5: offices for professional/commercial purposes (lawyers, accountants, GPs, government, architects) — but a medical practice where treatment leaves patients unconscious or non-ambulatory becomes a Class 9a health-care building.
- Class 6: shops, restaurants, cafés, hairdressers, public laundries, markets, showrooms, funeral parlours, shopping centres; service stations are Class 6, but premises solely for panel beating, auto-electrical or tyre replacement are Class 8.
- Class 7a: carparks; Class 7b: warehouses/storage/wholesale display. Class 8: factories, process/handicraft buildings, mechanics' workshops, abattoirs — and laboratories regardless of size, due to high fire hazard potential (labs within health-care buildings are instead Class 9a).
- Class 9a: health-care buildings (hospitals, clinics/day surgeries where treatment leaves patients unconscious/unable to move); Class 9b: assembly buildings (schools, universities, childcare, sporting facilities, nightclubs, public transport buildings); Class 9c: aged care buildings with personal care services and 24-hour staff assistance to evacuate.
- Class 10a: non-habitable buildings (sheds, carports, private garages); Class 10b: structures (fence, mast, antenna, retaining wall, swimming pool); Class 10c: private bushfire shelter — a structure associated with, but not attached to, a Class 1a building.
- A private garage is a garage associated with a Class 1 building, or a single storey containing not more than 3 vehicle spaces, or a separate single-storey garage of not more than 3 vehicles associated with another building.
- 10% ancillary rule: every part of a building must be classified separately, but a part with a different purpose occupying not more than 10% of the floor area of its storey may be treated as ancillary to the major use (e.g. an 8% office within a Class 8 warehouse stays Class 8; at 12% it must be classified separately as Class 5).
- A building or part may hold multiple classifications (e.g. Class 5/6/7) provided the design meets the minimum requirements of all of them; where classification is unclear, the approval authority has discretion to decide.
- A sole-occupancy unit (SOU) is a part of a building occupied by an owner/lessee/tenant to the exclusion of others — e.g. an apartment, a hotel suite, or a shop in a shopping centre.

## Caveats

- © 2017, referencing pre-NCC 2019 part numbers (Vol One Part A3, Vol Two Part 1.3); under NCC 2022 classifications live in Section A (A6G1–A6G11) of the Governing Requirements. Class definitions themselves are substantively unchanged, but verify edge cases against the current NCC.
- National perspective only — expressly does not address State or Territory variations.
