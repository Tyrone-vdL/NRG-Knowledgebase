# A&L Windows (Aluminium) — Light & Ventilation Chart

## Summary

A&L Windows reference chart listing light area (m²) and ventilation area (m²) for every item code in their aluminium window and door range — used to demonstrate compliance with NCC natural light and ventilation requirements (light typically ≥ 10% and ventilation ≥ 5% of room floor area for habitable rooms). Ranges covered: Awning (standard, overlooking, Windsor, Armadale, Georgian, Contempo, Contempo multi-transom, Metro, with-transom variants), Louvre, Sliding windows (including double-ended and double sash), Sliding doors, Entertainer sliding doors, Boutique French doors, and the Boutique window family (fixed no-mullion, 1/2/3 fixed mullions, Armadale, Metro, Georgian, multi-transom with sidelight variants).

## Key claims

- Each entry pairs an item code with Light (m²) and Vent (m²) — e.g. awning aa-1209: Light 0.82 m², Vent 0.90 m²; sliding window sw-1818: Light 2.93 m², Vent 1.14 m²; sliding door sd1-24: Light 4.41 m², Vent 2.29 m².
- Item codes encode nominal size (height/width digits, e.g. aa-1218 ≈ 1200 mm x 1800 mm class); suffix "f" denotes fixed (Vent = 0), "ol" overlooking, "des" double-ended sash, "ds" double sash, "fnm" fixed no mullion, "oi/oo" opening-in/opening-out for French doors.
- Fixed units (e.g. aa-0503f, btq1fnm series, st-xx03f) have **ventilation area = 0** — fixed glazing contributes light only.
- Louvre windows (wl- series) deliver ventilation areas nearly equal to their light areas (e.g. wl-2027: Light 6.37 m², Vent 6.28 m²) — the highest vent-to-light ratio in the range, useful for cross-ventilation strategies.
- Awning windows generally provide vent areas of roughly half to equal their light area depending on sash configuration; transom variants reduce vent area relative to equivalent plain awnings.
- Boutique French doors (btqfrd- series) provide vent areas larger than their light areas (e.g. btqfrd-2317oo: Light 2.73 m², Vent 3.57 m²) because the full door leaf opens.
- The chart contains several hundred entries across ~25 product families; it is a lookup table, not a performance (U-value/SHGC) document.

## Concepts touched

glazing, natural light, ventilation, NCC Housing Provisions Parts 10.5 and 10.6 (light and ventilation), window schedules, cross-ventilation, window open-ability, product-glazing

## Caveats

- Undated document; no NCC edition cited. A&L periodically updates ranges — confirm item codes against a current A&L price book before specifying.
- Pure tabular data; the PDF table extraction concatenated columns, so individual values transcribed above should be verified against the source PDF before use in a compliance report.
- Contains no thermal performance data (no U-values or SHGC) — pair with the A&L ComfortSmart Hybrid deck or WERS for energy ratings.
