# Telopea House Melbourne — Technical Drawings

## Summary

Architectural drawing set (single 12-sheet package) for the Telopea House two-storey reference design configured for Melbourne, covering site plan, ground and first floor plans, roof plan, four elevations, section, ground/first floor wall setout plans and a window/door schedule. Issued March 2020 at "Preliminary" project stage by the Commonwealth as a guide-only exemplar design; intended to be read alongside the companion Telopea House Melbourne "Design Options" document, which holds the building specifications and NatHERS rating information.

## Key claims

- Drawing set comprises 12 sheets: Site Plan (1:200), Ground Floor Plan, First Floor Plan, Roof Plan, four Elevations, Section, Ground Floor Wall Setout Plan, First Floor Wall Setout Plan (all 1:100), and Window and Door Schedule — A3 paper size, issued March 2020, project stage "Preliminary".
- Unlike the Hobart Telopea set (Options 1 and 2), the Melbourne set contains a single design version only.
- The drawings explicitly state they "should be read with the Telopea House Melbourne 'Design Options' document which includes building specifications and rating information" — thermal specs and star ratings are NOT in this drawing set.
- Window/door schedule: solid doors D1 2250mm x 1000mm (800mm eave), D2/D3 2250mm x 820mm; glazed hinged door HD1 2250mm x 920mm (600mm eave); glazed sliding doors SD2/SD3 2250mm x 2850mm (600mm eave); ground-level fixed/casement windows W2/W3 1050mm x 1500mm (W2 with a 600mm sun hood); W4 2250mm x 880mm fixed window with an external sliding sun screen; W5 600mm x 2850mm fixed/casement; upper-level fixed/casement windows W6–W11, W13, W14 at 1650mm x 1500mm with 600mm eaves; sliding windows W12 (1650mm x 880mm) and W15 (1650mm x 1340mm) both with external sliding sun screens plus 600mm eaves; W16 900mm x 600mm casement, frosted.
- The Melbourne schedule matches the Hobart Option 1 schedule (same opening references, dimensions and shading) — the climate-specific differentiation sits in the Design Options specification documents, not the drawing geometry.
- Shading is a mix of 600–800mm eaves, a 600mm sun hood, and external sliding sun screens on selected windows.
- Carries a Commonwealth disclaimer: plan is a guide only, not professional advice; no responsibility accepted for accuracy, fitness for purpose or regulatory compliance.

## Caveats

- These are architectural drawings — text extraction captures only title-block metadata and the window/door schedule. The actual plans, elevations, dimensions and annotations are graphical and must be viewed in the source PDF.
- Issued March 2020 at "Preliminary" stage, predating NCC 2022 7-star requirements; rating/specification claims live in the separate "Design Options — Telopea Melbourne" document.
- Guide-only Commonwealth exemplar; not site-specific or certified for construction. Window schedule rows were reconstructed from concatenated table text; verify exact dimensions against the source PDF.
