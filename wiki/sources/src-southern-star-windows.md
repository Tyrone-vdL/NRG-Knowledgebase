# Southern Star Windows Email — Fixed Window U-value/SHGC Enquiry

## Summary

A short email exchange (1 August 2018) between Matthew Morelli (NRG Efficient Homes) and Southern Star Windows internal sales about the U-value and SHGC of a double glazed clear aluminium FIXED window not listed on the WERS site. Southern Star's answer: use the awning window ratings for the fixed window, because both use the same frames.

## Key claims

- NRG (Matthew Morelli, matthew@nrgefficienthomes.com.au) asked for the U-value and SHGC of a Southern Star double glazed clear aluminium fixed window, noting WERS had no listing for that product.
- Southern Star Windows (Elliott Horvath, Internal Sales, 1 Aug 2018): "Refer to the awning window ratings for that as they use the same frames" — i.e. the fixed window may be rated using the awning window WERS values.
- No actual U-value or SHGC figures are given anywhere in the email — it is a substitution instruction only.
- Southern Star Windows contact at the time: 5 Kelly Court, North Geelong VIC 3215, ph 03 5277 7200.
- Enquiry related to postcode 3030 (Victoria), brochure type: aluminium.

## Concepts touched

glazing, windows, U-value, SHGC, WERS, NatHERS, aluminium frames, double glazing

## Caveats

- Informal email advice from August 2018, not a published WERS rating or test certificate — confirm current WERS listings before using awning values for fixed windows in an assessment.
- The advice is specific to Southern Star's aluminium range as at 2018; product ranges and ratings may have changed.
