# ABCB Advisory Note 2017-3 — Reference Building Verification Method V2.6.2.2 (misuse of NatHERS software)

## Summary

Four-page ABCB Advisory Note (first issued September 2017, revised October 2017, version 1.1) clarifying which calculation methods and energy analysis software may be used with the NCC Volume Two reference building Verification Method for houses, V2.6.2.2 — prompted by practitioners incorrectly using NatHERS software with that method. It warns that the specific input requirements of V2.6.2.2(c), especially temperature settings, zoning and overshadowing, are often not complied with, which can deliver houses substantially below the NCC's required energy efficiency level.

## Key claims

- The note's header labels it Advisory Note 2017-3 (despite the filename suggesting 2017-2); it applies to NCC Volume Two, Section 2, Part 2.6; first issued September 2017, revised October 2017.
- The ABCB had been advised that NatHERS software was, in some cases, being incorrectly used in conjunction with V2.6.2.2 — in particular the temperature settings and the unique features of the proposed house (e.g. zoning and overshadowing) were not being applied per the method's input requirements.
- It lists four basic options for complying with the NCC housing energy efficiency provisions at that time: (1) NatHERS-accredited software achieving the specified star rating (3.12.0(a)(i)); (2) the prescriptive DtS elemental provisions (3.12.0(a)(ii)); (3) the reference building Verification Method V2.6.2.2; (4) a Performance Solution other than V2.6.2.2.
- Unlike the star rating approach, V2.6.2.2 does not require any particular calculation method — any calculation method may be used provided it complies with the requirements listed in subclause V2.6.2.2(c).
- The Commonwealth Department of the Environment and Energy, as NatHERS Administrator, advised that NatHERS-accredited software only satisfies the V2.6.2.2(c) temperature settings in a limited number of circumstances.
- Incorrect application of V2.6.2.2 can result in houses delivering a level of energy efficiency substantially below that required by the NCC.
- The requirements must be read in conjunction with the building legislation of the relevant State or Territory; full guidance on V2.6.2.2 is in the ABCB handbook "NCC Volume Two Energy Efficiency Provisions 2016".

## Concepts touched

NCC, NCC 2016 (pre-2019 numbering), NatHERS, verification methods, reference building, Performance Solutions, climate zones, training

## Caveats

- Refers to pre-NCC 2019 clause numbering: V2.6.2.2 became H6V2 in NCC 2022 (with revised input requirements including specified thermostat settings) — the underlying warning (NatHERS software does not automatically satisfy the reference-building method's inputs) remains conceptually relevant, but cite current clauses.
- Filename/document mismatch: file is named "2017_2" but the document is titled Advisory Note 2017-3 — verify against the ABCB register if the note number matters.
