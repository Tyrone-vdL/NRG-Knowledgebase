# NCC 2022 First Principles Performance Solution Using Specification 44 — Appendix A copy

## Summary

This file is the ABCB "Understanding the NCC: First principles Performance Solution using Specification 44" guide (Version 2024.1, Aug 2024) re-issued with an "Appendix A" banner on every page — its body content is page-for-page identical to the main guide and it contains no additional appendix-specific material. It appears to have been filed as Appendix A to a project report or performance solution submission. For the actual content (H6P1 compliance pathways and the S44C2/S44C3/S44C4 load-limit formulas), use the primary page: [[src-ncc-2022-first-principles-performance-solution-using-spec-44]].

## Key claims

- Identical content to the main guide: NCC 2022 Part H6 (Volume Two) housing energy efficiency, Performance Requirements H6P1 (thermal performance) and H6P2 (energy use).
- H6P1 compliance options: DTS (Option 1 NatHERS, Option 2 Elemental), Verification Methods H6V2 (reference building) and H6V3 (envelope sealing), or a first-principles Performance Solution against the Specification 44 load limits.
- Specification 44 formulas restated: S44C2 HLL = greater of 4 MJ/m².annum and ((0.0044 × HDH) − 5.9) × FH; S44C3 CLL = (5.4 + 0.00617 × (CDH + 1.85 DGH)) × FC; S44C4 TLL combines HLL, CLL and Tr. All in MJ/m².annum.
- Performance Solution process per Clause A2G2(4): PBDB, analysis, evaluation, final report.
- The only difference from the main guide is the "Appendix A" label in the page header.
