# Acacia House Brisbane — Technical Drawings (Options 1 & 2)

## Summary

Architectural drawing set (two 10-sheet packages) for the Acacia House single-storey reference design configured for Brisbane, covering site plan, floor plan, roof plan, four elevations, section, wall setout plan and a window/door schedule for both Option 1 and Option 2. Issued March 2020 at "Preliminary" project stage by the Commonwealth as a guide-only exemplar design; intended to be read alongside the companion Acacia House Brisbane "Design Options" document, which holds the building specifications and NatHERS rating information.

## Key claims

- Drawing set comprises 10 sheets per option: Site Plan (1:200), Floor Plan (1:100), Roof Plan (1:100), four Elevations (1:100), Section (1:100), Wall Setout Plan (1:100), and Window and Door Schedule — all A3 paper size, issued March 2020, project stage "Preliminary".
- Two design options are provided for Brisbane (Option 1 and Option 2), each with its own complete 10-sheet set and window/door schedule.
- The drawings explicitly state they "should be read with the Acacia House Brisbane 'Design Options' document which includes building specifications and rating information" — thermal specs and star ratings are NOT in this drawing set.
- Window/door schedule (Option 1) lists 17 openings: main door D1 2400mm x 1100mm solid door; glazed sliding doors SD1–SD7 all 2400mm high (widths 2000–2300mm); fixed/awning windows W2/W3/W5 2400mm x 900mm; awning window W4 1440mm x 600mm; fixed window W6 800mm x 2300mm; sliding window W7 600mm x 600mm; glazed hinged door D2 2250mm x 820mm.
- Most openings are shaded by a 1000mm eave at 2700mm above Finished Floor Level (Brisbane uses deeper 1000mm eaves vs 600mm for the Hobart version of the same design — a climate-driven shading difference).
- Option 2 substitutes louvre windows for the Option 1 fixed/awning windows at W2, W3 and W5 (2400mm x 900mm) — increasing ventilation openings, consistent with a hot-humid climate cooling strategy; the rest of the schedule is otherwise near-identical.
- Carries a Commonwealth disclaimer: plan is a guide only, not professional advice; no responsibility accepted for accuracy, fitness for purpose or regulatory compliance.

## Caveats

- These are architectural drawings — text extraction captures only title-block metadata and the window/door schedules. The actual plans, elevations, dimensions and annotations are graphical and must be viewed in the source PDF.
- Issued March 2020 at "Preliminary" stage, predating NCC 2022 7-star requirements; rating/specification claims live in the separate "Design Options — Acacia Brisbane" document.
- Guide-only Commonwealth exemplar; not site-specific or certified for construction.
