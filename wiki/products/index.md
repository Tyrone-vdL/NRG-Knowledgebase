## How to use this catalog

1. User asks "what's the R-value of [product]?" → find product here, open its datasheet from source folder, read the value.
2. If the question recurs, file the answer back as `wiki/sources/src-<product-slug>.md` with the extracted figures so we don't have to re-read the PDF every time.

## Status

This is the **catalog** — file names organized by type. Detailed per-product summaries live in `wiki/sources/src-<slug>.md` and are filled in as needed.
