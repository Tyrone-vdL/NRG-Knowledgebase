# NatHERS — Nationwide House Energy Rating Scheme

NatHERS is the Australian government's framework for rating the thermal performance of residential buildings. Software like FirstRate5, AccuRate, and BERS Pro use NatHERS-accredited methodology.

## Why this page exists

Most product and design questions resolve to NatHERS in some way: "Will this glazing pass?" "What's the heating/cooling load?" "Does this fabric meet 7 stars?" The answer almost always means running an assessment in NatHERS-accredited software using the rules in the Assessor Handbook.

## Key sources (must-read order)

1. `NatHERS_Assessor_Handbook_2024.pdf` — current handbook, the primary rulebook for assessors
2. `NatHERS Technical Note 20241023.pdf` — most recent technical note (Oct 2024)
3. `nathers-heating-cooling-load-limits-2022.pdf` — load limits introduced 2022
4. `ABCB_Standard_NatHERS_heating_and_cooling_load_limits.pdf` — the ABCB standard the limits flow from
5. Guidance Notes folder (`Z - TECHNICAL INFO/NATHERS/GUIDANCE NOTES/`) — TPA responses, custom windows, edge cases
